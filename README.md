# Refined-Trump-Speech-Markov-Chain
A Markov chain generator with variable lengths and checks for proper sentence and quote ending. Dataset is from Trump's 2016 campaign speeches, but can be easily swapped out for something else.

Dataset: https://github.com/ryanmcdermott/trump-speeches
Inspired by https://medium.com/@jdwittenauer/markov-chains-from-scratch-33340ba6535b
